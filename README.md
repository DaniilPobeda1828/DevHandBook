## 1. ИНСТРУКЦИЯ К ПО

### 1.1 Назначение программы
Справочник для программистов, содержащий информацию по установке языков программирования, менеджеров библиотек и решению типичных ошибок.

### 1.2 Системные требования
| Компонент | Требование |
|-----------|------------|
| ОС | Windows 10/11 |
| Java | JRE 11 или выше |
| ОЗУ | 256 MB |
| Диск | 50 MB |

### 1.3 Запуск программы

**Способ 1 (EXE):**
- Дважды кликните на `DevHandbook.exe`

**Способ 2 (JAR):**
```bash
java -jar DevHandBook.jar
```

**Способ 3 (исходники в IDEA):**
- Открыть проект в IntelliJ IDEA
- Запустить `Main.java`

### 1.4 Работа с программой

| Действие | Как выполнить |
|----------|---------------|
| Поиск | Введите текст в поле "Поиск" |
| Копировать выделенное | `Ctrl+C` |
| Копировать всё | `Ctrl+Shift+C` |
| Фокус на поиск | `Ctrl+F` |
| Смена темы | Кнопка "Тёмная тема"/"Светлая тема" |
| Открыть документацию | Кнопка "Документация" |

### 4.4 Сборка проекта

**В IntelliJ IDEA:**
1. `Build` → `Build Artifacts` → `DevHandBook_jar` → `Build`
2. JAR появляется в `out/artifacts/DevHandBook_jar/DevHandBook.jar`

**В командной строке:**
```bash
javac -encoding UTF-8 src/*.java -d out/production/DevHandBook
jar cfe DevHandBook.jar Main -C out/production/DevHandBook .
```

**Создание EXE:**
- Использовать Launch4j
- JAR: `DevHandBook.jar`
- Main class: `Main`
- Output: `DevHandbook.exe`
