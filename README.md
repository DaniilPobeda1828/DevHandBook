## 1. ИНСТРУКЦИЯ К ПО

### 1.1 Назначение программы
Справочник для программистов, содержащий информацию по установке языков программирования, менеджеров библиотек и решению типичных ошибок.

### 1.2 Системные требования
| Компонент | Требование |
|-----------|------------|
| ОС | Windows 10/11 |
| Java | JRE 11 или выше (включена в папку `runtime`) |
| ОЗУ | 256 MB |
| Диск | 100 MB |

### 1.3 Структура распространения

```
DevHandBook/
├── DevHandBook.exe          # Главный исполняемый файл
├── app/                     # Папка с JAR и зависимостями
│   └── DevHandBook.jar      # Java архив приложения
├── runtime/                 # Встроенная Java Runtime Environment
│   └── (файлы JRE)
├── src/                     # Исходные коды (опционально)
└── README.md                # Документация
```

### 1.4 Запуск программы

**Способ 1 (рекомендуемый):**
- Дважды кликните на `DevHandBook.exe`

**Способ 2 (через JAR):**
```bash
java -jar app/DevHandBook.jar
```

**Способ 3 (исходники):**
```bash
javac -encoding UTF-8 src/*.java
java Main
```

### 1.5 Работа с программой

| Действие | Как выполнить |
|----------|---------------|
| Поиск | Введите текст в поле "Поиск" |
| Копировать выделенное | `Ctrl+C` |
| Копировать всё | `Ctrl+Shift+C` |
| Фокус на поиск | `Ctrl+F` |
| Смена темы | Кнопка "Тёмная тема"/"Светлая тема" |
| Открыть документацию | Кнопка "Документация" |
